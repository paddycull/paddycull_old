What we are

StudentLunchDealz is an application for students, which allows them to view nearby restaurants that have student lunch deals. The app is still in development, but the basic functionality is included in the map below.

So what's included?

As you can see, the map includes:
  • The nearby restaurants with student lunch deals.
  • The 'Staff Picks', which are given a gold star, as well as a title
  • The actual colleges in the area, which helps in locating the nearby restaurants.
  • The ability to search by area, when you open the map in full screen.

    • What's the plan for the future?

    Currently, the map only contains a few restaurants which I personally know the prices and locations of. This will soon be expanded upon, to include the locations of other colleges and restaurants in Dublin city.
